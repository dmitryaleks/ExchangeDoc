# docs/images/

Diagrams, screenshots, and tables captured from sources, referenced from the
topic Markdown files.

Naming convention: `{topic}_{short-description}.{ext}` — e.g.
`auctions_kospi_random_end.png`, `vi_thresholds_table.svg`. Keep filenames
lowercase and use underscores; this avoids cross-platform path issues.

Prefer SVG for diagrams, PNG for screenshots. If a screenshot is taken from a
PDF or web page, archive the source itself in `../sources/` so the image can
be reproduced if needed.
