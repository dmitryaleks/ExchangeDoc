# docs/kospi/

KOSPI-specific topic write-ups.

One Markdown file per topic, named to match the entry in `../index.yaml`
(e.g. `market_hours.md`, `auctions.md`, `price_ranges.md`).

Topics that are identical across KOSPI and KOSDAQ should live in `../common/`
and be referenced from `../index.yaml` with `markets: [KOSPI, KOSDAQ]` instead
of being duplicated here.

See `../STYLE.md` (to be written in Phase 0.3) for the per-file template.
