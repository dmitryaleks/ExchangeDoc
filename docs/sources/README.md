# docs/sources/

Archived primary sources (PDFs, HTML snapshots) plus an `INVENTORY.md` index.

Each source is assigned a stable ID (e.g. `KRX-BR-2024-09`) used in
`../index.yaml`'s `sources:` field and in inline citations from topic files.

`INVENTORY.md` (to be written in Phase 1.4) records, per source:

  - id, title, publisher, URL, fetch date, language, version / effective date,
    archived filename, topics it covers.

Prefer archiving the original-language version (often Korean) alongside any
English translation, especially when a numeric threshold is load-bearing.
